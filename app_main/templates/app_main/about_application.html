What is CloudyContacts and how it works

Web application built with the Django framework that provides users with tools to manage contacts and notes, upload files to cloud storage, and receive news summaries. The application includes authentication mechanisms to ensure the privacy and security of user data.

User authentication

The application has an authentication mechanism:
  • the web interface provides access to its functions only to registered users;
  • each registered user has access only to their own data and files;
  • implemented a password recovery mechanism for users by email.
    • Data is validated during the registration of a new user.

Contacts

An authorized user using the application can:
  • save contacts with names, addresses, phone numbers, emails, and birthdays to the contact book;
  • display a list of contacts whose birthday is within a specified number of days from the current date;
  • search for contacts among the contacts in the directory;
  • edit and delete entries from the contact book.
    • The application validates the contact data when creating or editing an entry and notifies the user in case of incorrect entry.

Notes

An authorized user using the application can:
  • store notes with text information;
  • search notes;
  • edit and delete notes;
  • add 'tags' to notes (keywords describing the topic and subject of the record);
  • search and sort notes by keywords (tags).

Files

An authorized user using the application can:
  • upload files to the cloud service and access them;
  • sort files by category (images, documents, videos, etc.);
  • filter files by the selected category.

News

An authorized user using the application can:
  • View a summary of the day's news. Namely, select one of the proposed categories of news (sports, politics, or finance), and the application will show the headlines of the latest news, with the ability to go to the resource with the news.
